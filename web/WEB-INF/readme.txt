Всё, что лежит в папке WEB-INF, недоступно для пользователя. То есть он
не может ввести в браузере путь к файлу из этой папки и открыть этот файл.
Сюда можно класть те страницы, к которым юзер не должен имть доступ, но которые
используются другими страницами/сервлетами (например header-ы и footer-ы).
