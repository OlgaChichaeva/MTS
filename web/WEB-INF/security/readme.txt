WEB-INF -> SECURITY

Пакет для облегчения разграничения доступа к страницам.

Как пользоваться:
На той странице, к которой требуется ограничить доступ, сначала
делается include страниц с названием вида "accept%имя_роли%",
где %имя_роли% - название того типа пользователей, доступ
которым нужно разрешить.

Затем делается include файла checkAccept, который, если пользователю
запрещен доступ к странице, выдаёт соответствующее сообщение.

Пользователю admin по умолчанию разрешён доступ.

Пример:

<jsp:include page="acceptLegalEntity.jsp" />
<jsp:include page="checkAccept.jsp" />

Доступ будет только для юр. лиц и админа (по умолчанию).
